# 🏢 AI-Powered HR ERP & Smart Retention Portal

An advanced End-to-End Machine Learning and Decision-Support System designed to predict employee attrition risk and calculate cost-effective, merit-based retention strategies to prevent corporate loss.

## 🚀 Key Features
- **Live AI Risk Assessment:** Powered by a trained Random Forest Classifier to catch high-risk employees early.
- **Corporate Loss-Control Optimizer:** Evaluates employee performance metrics (Skills Mastery, 90%+ Attendance, Domain Knowledge) before formulating counter-offers.
- **Interactive HR ERP Database:** In-memory secure state to register multiple employees (scaled up to $500,000 salary limit) and manage profiles.
- **Smart Profile Search Engine:** Instantly pull up an employee's complete profile, address, blood group, and live AI insight using a unique Employee ID.

- **App/app.py:** Streamlit Web Application Code
- **Notebooks/Models/attrition_rf_model.pkl:** Trained Random Forest Model File
